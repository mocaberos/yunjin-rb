### 使用方法
```
gem 'yunjin-rb'
```

### 動作確認(irb)
```shell
$ ./bin/console
# or
$ docker-compose run yunjin_rb
```

### テストを実行する
```shell
$ ./bin/test.sh
# or
$ docker-compose run yunjin_rb /app/bin/test.sh
```
